Valid Anagram (Easy)

Problem

Given two strings `s` and `t`, return True if `t` is an anagram of `s`, and False otherwise.
An anagram is a word formed by rearranging the letters of another, using all original letters exactly once.

Examples

Input:

s = "anagram"

t = "nagaram"

Output: True

---
Input:

s = "rat"

t = "car"

Output: False
